My reaction to Git

To me, Git is both easy and hard to use. At first, I was intimidated by it, but it got easier after a while. The easiest part about Git is typing things into Terminal. While getting Terminal to actually run the commands is hard, the hardest part is actually pushing things into the repository. In fact, I was unable to finish it due to errors constantly popping up when trying to push. Terminal is easy to use, but it has many issues and errors which is frustrating. The errors are the most confusing thing about Git.
